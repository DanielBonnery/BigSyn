
BigSyn
======

Installation
------------

`devtools::install_github("DanielBonnery/BigSyn")`

Package dependencies
--------------------

installed.packages()\["BigSyn","Depends"\] `utilities::getDependencies("BigSyn")`

Demo
----

To run the demo, just run: `demo(Synthesize_database)`
